## Risk Analysis (TAME Framework)

**Time**
The main time risk is underestimating how long it takes to write content and style two pages consistently. To manage this, I will build both HTML pages before touching the CSS.

**Authority**
This is a solo project, so there are no approval bottlenecks. All decisions are mine.

**Money**
All tools used are free, including GitHub Pages, VS Code, and plain HTML and CSS.

**Environment**
The site needs to work in a standard browser.
