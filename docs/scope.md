# PROJECT SCOPE STATEMENT

**Project Name:** Berry Coffee Shop Website
**Completed by:** Casey Trejo
**Projected Start:** July 12, 2026

---

## PROJECT PURPOSE:

To apply the full project management cycle from *Project Management for the Unofficial Project Manager* to a real technical deliverable. This scope statement will serve as the guiding document for designing and building a website for Berry Coffee Shop, a fictional local coffee shop.

---

## PROJECT DESCRIPTION:

This project involves building a two-page website for Berry Coffee Shop using HTML and CSS. The site will include a home page (`index.html`), an about page (`about.html`), and a stylesheet (`style.css`).

---

## DESIRED RESULTS:

- A public GitHub repository
- A functioning two-page website with:
  - `index.html` — home page introducing Berry Coffee Shop, its menu highlights, and atmosphere.
  - `about.html` — a page describing the shop's story, values, and team.
  - `style.css` — a single shared stylesheet applied to both pages.

---

## EXCLUSIONS:

- No real business information — this is a fictional project

---

## COMMUNICATION NEEDS:

Solo project.

---

## ACCEPTANCE CRITERIA:

The project is complete when:

- All files are committed to a public GitHub repository
- The live site is accessible via GitHub Pages
- All three documentation files are present in the `docs/` folder
- Submitted through Canvas

## CONSTRAINTS:

- Must use only HTML and CSS
- Must be hosted on GitHub Pages
